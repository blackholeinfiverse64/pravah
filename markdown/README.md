# Phase 6 Adversarial Test Suite

This suite targets replay tampering, ordering corruption, unsigned event injection,
dependency failures, concurrent replay reconstruction, and deterministic recovery drift.

Planned coverage:
- Tampered replay rejection
- Ordering corruption rejection
- Unsigned event rejection
- Dependency failure handling
- Concurrent replay determinism
- Deterministic recovery validation

The initial files are skeletons only. Each scenario will be implemented and validated one attack at a time.
