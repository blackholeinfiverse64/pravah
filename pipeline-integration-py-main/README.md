# Pipeline Dashboard

Modern web-based dashboard for monitoring pipelines.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the dashboard:
```bash
python dashboard.py
```

3. Open browser and go to:
```
http://localhost:5000
```

## Features
- Real-time statistics
- Interactive charts
- Auto-refresh every 5 seconds
- Modern dark theme UI
